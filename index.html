What is the icosaidrium?

The icosaidrium is a map editor from all devices avaliable in germany, portuguese (BR), englsh (EUA), fanch and spansh, was devloped to help to help designers and devlopers to make maps.

Features

  • Works in alls devices
  • Is fast and not takes much space
  • Avaliable in five languages
  • Costumizable
  • Simple use

Getting started

How to create a new map?

To create a new map just enter on the apk and click in "Map" after click in "New / Load Map" and insert the map name and the grid size, if the map exists its will load it else it will create a new empety map to you.

How to insert my tileset?

After create your map you need insert a tileset right? to do it go to the file in your system that matches with 'android/data/org.icosaidrium/maps/(your map name)/tileset' and insert your 'tileset.png' and 'tileset.json' with this same name and extension type.

How to render my map in my game?

Now you finish your map and need render it in your game, do it is simple go to the map on icosaidrium files 'android/data/org.icosaidrium/maps/(your map name)', copy the file 'mapbatch.json' to the game project and use the icosaidrium parser: Icosaidrium parser

Buttons (mobile only)

  • M: Enable move screen mode
  • S: Save the map
  • B And N: Remove or adds id on the object id
  • R: Rotate the block
  • Y and Z: Undo and redo
  • V and H: Mirror in the vertical and horizontal
  • T: Enable the layer editor mode
  • Alt: Change the mode (clicks multiples times to change the mode)
  • Backspace: Enable you eraser blocks or objects (click multiples times to change the mode)
  • T + Backspace: Delete all the layer
  • Esc: Back to main menu
